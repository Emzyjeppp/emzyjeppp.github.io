# Muhammad Jepri | Portfolio

Personal minimalist portfolio website built with HTML, CSS (Vanilla), and JavaScript.

## Features
- **Minimalist & Clean Layout**: Focuses on professional readability and performance.
- **Light/Dark Mode Toggle**: Supports instant theme switching with state persistence.
- **Project Detail Modals**: Interactive popup modals for detailed project previews.
- **Education & Experience Timeline**: Displays academic progression (D3 RPLA and S1 Informatika) at UTDI.
- **Fully Responsive**: Mobile-first grid layouts for seamless scaling.

## Structure
- `index.html`: Webpage structure.
- `style.css`: Design system and custom styles.
- `script.js`: Theme switching, modal interactions, active link highlighting, and email copy helper.
